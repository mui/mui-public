# Components

## Code Highlighting

- [`CodeHighlighter`](./code-highlighter/page.mdx): A component for displaying code snippets with syntax highlighting.
- [`CodeController`](./code-controller/page.mdx): A component for controlling the code displayed
- [`HighlightProvider`](./highlight-provider/page.mdx): A component that provides client-side functions for fetching and highlighting source code.
