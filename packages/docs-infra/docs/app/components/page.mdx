# Components

## Code Highlighting

- [`CodeHighlighter`](./code-highlighter/page.mdx): A component for displaying code snippets with syntax highlighting.
- [`CodeControllerContext`](./code-controller-context/page.mdx): A component for controlling the code displayed
- [`CodeProvider`](./code-provider/page.mdx): A component that provides client-side functions for fetching and highlighting source code.
