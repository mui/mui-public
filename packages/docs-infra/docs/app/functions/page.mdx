# Functions

Developer utilities and build-time tools for the documentation infrastructure.

## Available Functions

- [`loadPrecomputedCodeHighlighter`](./loadPrecomputedCodeHighlighter/page.mdx) - Webpack loader for build-time demo optimization
- [`transformHtmlCode`](./transformHtmlCode/page.mdx) - Plugin for transforming HTML code blocks into precomputed data structures
- [`transformMarkdownCodeVariants`](./transformMarkdownCodeVariants/page.mdx) - Remark plugin for transforming markdown code blocks with variants into HTML structures
- [`loaderUtils`](./loaderUtils/page.mdx) - A collection of utilities for loading and processing code, including parsing imports and resolving module paths
