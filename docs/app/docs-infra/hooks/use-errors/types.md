## API Reference

### useErrors

Provides access to error state in an isomorphic error handling system.
Implements the Props Context Layering pattern to work seamlessly across
server and client boundaries.

**useErrors Parameters:**

| Parameter | Type     | Default | Description                                                    |
| :-------- | :------- | :------ | :------------------------------------------------------------- |
| props     | `Errors` | -       | Optional props containing fallback errors (typically from SSR) |

**useErrors Return Value:**

Object containing the current errors array (context errors take precedence over props)

| Property | Type                   | Description |
| :------- | :--------------------- | :---------- |
| errors   | `Error[] \| undefined` | -           |
