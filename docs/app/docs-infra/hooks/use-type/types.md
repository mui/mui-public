## API Reference

### TypesDataProvider

Provider that collects type data from all `useTypes()` calls within its subtree.
This enables `useType(name)` to look up individual type data anywhere in the tree.

**TypesDataProvider Props:**

| Prop       | Type              | Default | Description |
| :--------- | :---------------- | :------ | :---------- |
| children\* | `React.ReactNode` | -       | -           |

### useType

Hook to look up a single type's data by name from the nearest `TypesDataProvider`.

Returns the `TypeData` for the given name, or `undefined` if:

- No `TypesDataProvider` is present in the tree
- The type name has not been registered

**useType Parameters:**

| Parameter | Type     | Default | Description                                                            |
| :-------- | :------- | :------ | :--------------------------------------------------------------------- |
| name      | `string` | -       | The type name to look up (e.g., "Root", "Trigger", "AccordionTrigger") |

**useType Return Value:**

```tsx
type ReturnValue = TypeData | undefined;
```

### useTypesDataContext

Returns the TypesDataContext value, or undefined if not within a provider.
Used internally by `useTypes` to register types and by `useType` to look up types.

**useTypesDataContext Return Value:**

```tsx
type ReturnValue = TypesDataContextValue | undefined;
```
